# GifToCmd
将gif在命令行展示出来。可以扩展为播放任意gif 喜欢的点个star(￣▽￣)"

## 编译
```shell
git clone https://github.com/Coder-Qing/GifToCmd.git
cd cxk
go build -o /path/executablefile ./*.go
```
